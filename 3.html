대사란, 산소 + 조절 영양소(비타민,미네랄) 의 조합으로 칼로리 영양소가 활활 타는 것 입니다.
오랫동안 절식 다이어트를 한 후에 대사량이 감소하는 원인은 1. 근육량 감소 2. 조절영양소 부족(바닥)입니다.
먹는 게 없으니 아무리 산소가 있어도 탈것이 없는 거죠.
그러니 자연스레 대사의 능력도 떨어져 버려요.

대사량 높이는 방법 : 식사


1.근육량을 키우는 식사법! ( 저탄 고지 식단으로! )

탄수화물, 단백질, 지방 등이 다들 칼로리를 내는 에너지인 건 아시죠? 그럼 이제 고민에 빠지죠.
과연 내가 칼로리를 탄수화물로 채울 것인가, 단백질로 채울 것인가, 지방으로 채울 것인가, 여러가지 이야기들도 있구요 ㅠㅠ
건강하게 근육량을 증가시키는 방법은 ! 저탄고지 ( 단백질이 포함된 어육류를 많이 섭취하고, 또 근육은 지방을 에너지원으로
사용하는 것을 너무너무 좋아해요. )


/ 느린대사 저탄고지 식단 /

  • 퀴노아+귀리+수수밥 끼니당 50g ( 계란한개 정도 )
  • 한끼 고기량 150g
  • 익힌 야채(오일만주스) 끼니 1~2컵씩 or 오일에 볶은 잎채소및 야채들 넉넉히


2. 조절영양소는 무엇으로 채워야하나?

충분히 익힌 야채! 익힌 채소를 갈아먹으면( 1~2컵/일 ) 빠르게 조절영양소 채움!
오랜 절식다이어트를 한 사람들은 소화능력이 많이 떨어져있어요. 익힌 야채는 소화력도 높여줘요!

대사량 높이는 방법 : 생활 습관


절식 다이어트를 오래한 사람들은 식사만으로는 빠르게 대사량을 복원하기 쉽지 않아요. 영양제를 보충해야만해요.
1. 비타민B군 (B1,B2,B3,B5,B6,B12) 이 대부분의 대사과정의 도우미역할!
